This is a SolidJS client only application. It has no backend and only runs in the browser.

It uses SolidJS with Typescript for the frontend and TailwindCSS for the styling. It uses solid-router for routing. Use `bun` as the package manager and javascript runtime.

Page:
- Home: Links to the utilities
- PDF Merge: A page that has a file drop zone, shows the list of files selected, allows for reordering and removal of files, and a button to merge the files. The merged file is downloaded to the user's device.
- VTT Converter: A page that has a file drop zone at the top. When a .vtt is dropped, read the file and show it in a conversation transcript format.

# VTT Converter
```
Person 1: Hello, how are you?
Person 2: I'm good, thank you. How are you?
Person 3: Let's start the meeting.
```

Color code each participant with a different color.

You may request or suggest packages to install that will help you with the task. Before you install a new package, you MUST ask the user for permission.
